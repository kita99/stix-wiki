# Artifact

The Artifact object permits capturing an array of bytes (8-bits), as a base64-encoded string, or linking to a file-like
payload.

One of payload_bin or url MUST be provided. It is incumbent on object creators to ensure that the URL is accessible for
downstream consumers.
